# 📚 Introdução à Programação

## 💡 O que é Programação?

Programação é a **resolução de problemas por meio de instruções** que passamos para um computador executar.

Essas instruções permitem que dispositivos como:
- Celulares 📱
- Computadores 💻
- Televisões 📺

executem aplicações que usamos no dia a dia.

### Exemplos práticos:
- Aplicativos de entrega de comida
- GPS e navegação
- Plataformas de streaming
- Redes sociais

➡️ Tudo isso funciona graças à programação.

---

## 🗣️ Linguagens de Programação

Assim como seres humanos utilizam idiomas para se comunicar, computadores precisam de **linguagens de programação** para entender as instruções.

Cada linguagem é mais adequada para determinados tipos de problemas.

### 🔹 Exemplos de linguagens:

#### 🐍 Python
- Simples e fácil de aprender
- Muito usada em:
  - Ciência de dados
  - Automação
  - Scripts

#### ☕ Java
- Mais robusta e estruturada
- Muito utilizada em:
  - Sistemas corporativos
  - Grandes aplicações
- Oferece maior controle e segurança

#### 🌐 JavaScript
- Presente na maioria dos sites
- Usada para:
  - Desenvolvimento web
  - Interatividade em páginas

---

## 🧠 Lógica de Programação

Antes de aprender qualquer linguagem, é fundamental entender a **lógica de programação**.

### O que é lógica?

É a capacidade de:
- Entender um problema
- Quebrá-lo em partes menores
- Criar uma sequência de passos para resolvê-lo

---

## 📌 Algoritmos

Um **algoritmo** é uma sequência de passos para resolver um problema.

### Exemplo simples:
1 Pegar um copo
2 Encher com água
3 Beber


➡️ Isso já é um algoritmo.

### Por que isso é importante?

- O algoritmo é **independente da linguagem**
- Você pode escrever a lógica uma vez
- E depois aplicar em qualquer linguagem (Python, Java, etc.)

---

## 🚀 Por onde começar?

### Ordem recomendada:

1. Aprender lógica de programação
2. Praticar criação de algoritmos
3. Escolher uma linguagem
4. Aprofundar na linguagem escolhida

---

## 🎯 Conclusão

A programação não começa com código, mas com **pensamento lógico**.

Ao dominar a lógica:
- Você consegue aprender qualquer linguagem
- Resolve problemas com mais facilidade
- Evolui muito mais rápido na área
