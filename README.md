## 💡 O que é Programação?

Programação é a **resolução de problemas por meio de instruções** que passamos para um computador executar.

Tudo o que usamos no dia a dia envolve programação:

- Aplicativos de comida
- GPS
- Streaming
- Redes sociais

➡️ Programar é ensinar o computador a resolver problemas.

---

## 🧠 Lógica de Programação (O mais importante)

Antes de aprender qualquer linguagem, você precisa dominar a **lógica de programação**.

### O que é lógica?

É a capacidade de:
- Entender um problema
- Quebrá-lo em partes
- Criar uma sequência de passos para resolvê-lo

---

## 📌 Algoritmos

Um algoritmo é um passo a passo para resolver um problema.

### Exemplo:
1 Pegar um copo
2 Encher com água
3 Beber


➡️ A lógica é independente da linguagem.

## 🛠️ Ferramentas Importantes

### 💻 Cisco Packet Tracer
Ferramenta de simulação de redes.

Permite:
- Criar redes virtuais
- Configurar roteadores e switches
- Simular tráfego de dados
- Testar conectividade

➡️ Essencial para aprender redes na prática.

---

### 🔍 Wireshark
Ferramenta de análise de tráfego de rede.

Permite:
- Capturar pacotes em tempo real
- Analisar protocolos
- Identificar problemas de rede
- Entender como os dados trafegam

➡️ Mostra o que acontece "por trás" da comunicação.
