# 🧭 Áreas da Programação

Após aprender **lógica de programação**, o próximo passo é entender as principais áreas de atuação.
Isso ajuda a escolher um caminho mais claro e definir quais tecnologias estudar.

---

## 🌐 Front-end

O **Front-end** é a parte visual de um sistema, ou seja, tudo aquilo com que o usuário interage diretamente.

### Exemplos:
- Sites
- Interfaces de sistemas
- Botões, formulários e menus

### Tecnologias utilizadas:

- **HTML** → Estrutura do conteúdo
- **CSS** → Estilização e design
- **JavaScript** → Interatividade

### Frameworks populares:
- React
- Angular

➡️ O foco do front-end é **experiência do usuário (UX/UI)**.

---

## ⚙️ Back-end

O **Back-end** é a parte responsável pelo processamento das informações.

É onde acontece a lógica do sistema, validações e comunicação com banco de dados.

### Exemplo prático:
Ao realizar uma transferência bancária:
1. O sistema recebe os dados
2. Verifica saldo
3. Valida a conta de destino
4. Atualiza as informações no banco de dados

### Tecnologias utilizadas:

- **Java**
- **JavaScript (Node.js)**
- **C#**

### Banco de dados:
- SQL (Structured Query Language)

➡️ O foco do back-end é **regra de negócio, segurança e processamento de dados**.

---

## 📱 Desenvolvimento Mobile

O desenvolvimento **Mobile** é voltado para aplicativos de celular.

Assim como o front-end, envolve interface e interação com o usuário, mas focado em dispositivos móveis.

### Plataformas:

- Android
- iOS

### Tecnologias nativas:

- **Kotlin** → Android
- **Swift** → iOS

### Desenvolvimento multiplataforma:

- **Flutter**
  - Linguagem: Dart
  - Permite criar um único código para Android e iOS

➡️ Boa opção para iniciantes que querem entrar no mundo mobile.

### 🤖 Inteligência Artificial
- Criação de sistemas inteligentes

## 🎯 Conclusão

A área de programação é ampla e cheia de possibilidades.

O mais importante é:
- Entender o básico (lógica)
- Explorar as áreas
- Escolher um caminho
- Praticar constantemente

➡️ Não existe escolha perfeita, existe **consistência**.
